## ğŸš€ How to Deploy to GitHub Pages

### Step 1: Create a GitHub Repository

1. Go to [GitHub.com](https://github.com) and sign in
2. Click the **"+"** button in the top-right corner
3. Select **"New repository"**
4. Name your repository (e.g., `mood-matching-game`)
5. Make sure it's set to **Public**
6. **Do NOT** check "Add a README file"
7. Click **"Create repository"**

### Step 2: Upload Your Files

You'll have all these files:
```
index.html
README.md
logo.png
card_back.png
mood_bored.png
mood_worried.png
mood_excited.png
mood_cool.png
mood_calm.png
mood_sick.png
```

**Upload via GitHub Website (Easiest)**

1. On your new repository page, click **"uploading an existing file"**
2. Drag and drop ALL 10 files (index.html, README.md, and all 8 .png images)
3. Add a commit message like "Initial game upload"
4. Click **"Commit changes"**

### Step 3: Enable GitHub Pages

1. Go to your repository on GitHub
2. Click **"Settings"** (top menu)
3. Click **"Pages"** in the left sidebar
4. Under "Source", select **"Deploy from a branch"**
5. Under "Branch", select **"main"** (or "master")
6. Make sure the folder is set to **"/ (root)"**
7. Click **"Save"**

### Step 4: Access Your Game

1. Wait 1-2 minutes for GitHub to build your site
2. Your game will be available at:
   ```
   https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/
   ```
   
   For example: `https://jsmith.github.io/mood-matching-game/`

3. You can find the exact URL at the top of the Pages settings

### Step 5: Share!

Send the link and play on any iPad browser! The game works great in Safari and Chrome.
